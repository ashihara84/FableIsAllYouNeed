## 第1章 確率の言葉 — 不確かさを数にする

- 1.1 動機: 分類モデルの出力は「答え」ではなく「自信」であるべき(第3巻エピローグの 0〜1 の正体)
- 1.2 確率の基本ルール: 全部足すと1、0以上(公理は最小限に)
- 1.3 確率変数と離散分布。例は一貫して「次の単語当て」(サイコロではなく言語で動機づけ。第6巻への地ならし)
- 1.4 条件付き確率と同時確率(言語モデル P(次の単語 | これまでの単語) を読み書きするのに必要な分だけ)
- 演習: 小さな文章コーパスで単語の出現確率を数える

## 第2章 期待値と分散 — 分布を2つの数で要約する

- 2.1 期待値: 確率で重みづけた平均
- 2.2 分散と標準偏差: ばらつきの量
- 2.3 和の期待値・和の分散: 独立なら足せる(**√d_k の伏線。この性質が終盤で効く**と予告)
- 2.4 **[コード]** サンプリングで期待値・分散を体感(大数の法則を図で)
- 演習: 手計算 + シミュレーションで一致確認

## 第3章 尤度と最尤推定 — 「もっともらしさ」を最大化する

- 3.1 逆向きの問い: データからパラメータを当てる
- 3.2 尤度関数: パラメータの関数として見た「データの出やすさ」
- 3.3 対数尤度: 積を和に変える(数値的にも理論的にも)
- 3.4 コイン投げ(ベルヌーイ)の最尤推定を手で解く: 答えが「頻度」になる気持ちよさ
- 3.5 視点の合流: 「尤度最大化」も「損失最小化」も同じ形 — 負の対数尤度という損失
- 演習: 最尤推定を微分(第2巻の道具)で解く

## 第4章 ロジスティック回帰 — 宿題の回収(クライマックスその1)

- 4.1 シグモイドの正体: 出力を「確率」として正当化する(log-odds からの導出は軽く、機能の理解を主に)
- 4.2 ベルヌーイ尤度 → 負の対数尤度 = **log loss**。分類の正しい損失が「導出されて」出てくる
- 4.3 勾配を手で出す: sigmoid + log loss の勾配は驚くほどきれい(y − t)・x
- 4.4 **[コード]** ロジスティック回帰をフルスクラッチ実装し、**第3巻エピローグと同じデータで学習が進む**ことを確認。MSE 版との学習曲線比較(勾配が死なない図)
- 4.5 なぜ MSE では勾配が死んだのか: 勾配式の比較で種明かし
- 演習: 決定境界の可視化

## 第5章 情報量とエントロピー — 驚きを測る

- 5.1 情報量 = 驚きの量: −log p という定義の妥当性(稀なほど驚く、独立なら足せる)
- 5.2 エントロピー: 驚きの期待値。分布の「不確かさ」
- 5.3 cross-entropy: 本当の分布 p を、自分の分布 q で符号化する損
- 5.4 KL divergence: cross-entropy − entropy。分布間の「ズレ」
- 5.5 **導出のクライマックス: 尤度最大化 = cross-entropy 最小化**(第4章の log loss が cross-entropy の特殊例だったと判明する構成)
- 演習: 小さな分布で entropy / cross-entropy / KL を手計算

## 第6章 softmax — 多クラスへ

- 6.1 2クラスから K クラスへ: 「スコアの列を確率分布に変える」関数が欲しい
- 6.2 softmax の定義と性質: exp で正に、和で割って1に。最大値シフトによる数値安定化
- 6.3 softmax + cross-entropy の勾配もきれい: (y − t)(第4章ときれいに対応)
- 6.4 **[コード]** 多クラス分類(softmax回帰)をフルスクラッチ実装
- 6.5 温度: 分布を尖らせる・なだらかにする(√d_k への前振り)
- 演習: one-hot と cross-entropy の関係を確認

## 第7章 内積の分散と √d_k — 伏線回収

- 7.1 問い: 次元 d_k のベクトル同士の内積は、どれくらいの大きさになるか
- 7.2 第2章の道具で計算: 成分が平均0・分散1で独立なら、内積の分散は d_k(脚注4の再現)
- 7.3 大きすぎるスコアを softmax に入れると何が起きるか: 分布が尖り、勾配が死ぬ(**第3巻エピローグと同じ病気**)
- 7.4 **[コード]** d_k を変えて内積のヒストグラム → softmax 後の分布 → 勾配の大きさ、を観察。√d_k で割ると治ることを確認
- 7.5 perplexity: エントロピーの指数。言語モデルの「平均分岐数」(本格的な出番は第6巻と予告)
- 演習: 脚注4を自分の言葉で書き直す
