# 第2章 最小値と傾き — 解けない方程式だらけの世界

> [目次](../TOC.md) ・ [← 前の章](01-derivative.md) ・ [次の章 →](03-gradient-descent-1d.md)

前の章で、私たちは微分という道具を手に入れました。関数の任意の点で「そこでの変化の速さ」を傾きという1つの数として取り出せます。多項式なら解析的なルールで、思い出せなければ数値微分で取り出せます。傾きは、もう測れます。

何のために傾きが欲しかったのでしょうか。第1巻で読めるようになった $XW + b$ の $W$ をどう決めるか——「$W$ をどっちに動かせば良くなるか」を知りたい、それが第1章の動機でした。

ただし「良くなる」は曖昧すぎます。この巻では、これを単純な形に置き換えます。**ある関数 $f(x)$ の値を、できるだけ小さくしたいのです。** 「良さ」をどう1つの数に押し込め、その数を何と呼ぶかは第3巻の仕事です。いまは「小さくしたい数があり、それは $x$ を動かすと変わる」という設定だけを受け取ってください。

つまり問題は**最小値(minimum)探し**になりました。この章でわかるのは3つです。最小値と傾きにははっきりした関係がある(2.1節)。その関係を方程式として解こうとすると、世の中のほとんどの関数で行き詰まる(2.2節)。だから、解くのをやめて、近づく(2.3節)。この3つめの転換が、巻全体の主役です。

## 2.1 最小値では傾きが 0

まず、いちばん見やすい関数で観察します。

$$f(x) = (x - 2)^2 + 1$$

展開すれば $f(x) = x^2 - 4x + 5$ なので、第1章のルールで微分できます。

$$f'(x) = 2x - 4 = 2(x - 2)$$

いくつかの点で、関数の値と傾きを並べてみましょう。

| $x$ | $f(x)$ | $f'(x)$ | 様子 |
|---|---|---|---|
| 0 | 5 | $-4$ | 急な下り坂 |
| 1 | 2 | $-2$ | ゆるい下り坂 |
| 2 | 1 | $0$ | **平ら** |
| 3 | 2 | $+2$ | ゆるい上り坂 |
| 4 | 5 | $+4$ | 急な上り坂 |

グラフに重ねると、こうなっています。

```
 f(x)
   │
 5 ┤ ●                            ●
   │  ＼                         ／
   │   ＼ 傾き −4       傾き +4 ／
 2 ┤     ●                    ●
   │      ＼ 傾き −2  傾き +2／
 1 ┤         ●──────────●
   │            傾き 0
   └──┬────┬────┬────┬────┬──→ x
      0    1    2    3    4
```

図2.1: $f(x) = (x-2)^2 + 1$ と各点での傾き。谷底に向かって左側は傾きが負(下り)、右側は正(上り)、そして谷底 $x = 2$ ではちょうど 0(平ら)。谷底から離れるほど傾きの絶対値は大きく、坂は急になる。

この図が言っているのは1つです。

**最小値の足元では、傾きが 0 になります。**

裏側からも確かめておきます。もし最小値の点で傾きが 0 でなかったとして、たとえば傾きが負(下り坂)なら、もう少し右に進めば $f$ はさらに小さくなる——つまりそこは最小値ではありません。傾きが正なら同じ議論が左向きに成り立ちます。最小値の点で許される傾きは 0 しか残りません。

これで、「最小値を探せ」が「傾きが 0 になる点を探せ」に化けました。$f'(x)$ は第1章のおかげで式として書けるので、あとは

$$f'(x) = 0$$

という**方程式**を解けばよいのです。先ほどの例なら $2(x-2) = 0$ から $x = 2$。中学の一次方程式です。最小値探しは、方程式を解く問題に帰着した——ように見えます。

ただし注意が要ります。示したのは「最小値ならば傾き 0」であって、逆ではありません。**傾きが 0 でも、そこが最小値とは限らない**のです。傾き 0 の点(**停留点**(stationary point))には、谷底のほか、山頂(動くとどちらでも下がる)や踊り場(一瞬平らになり、また同じ向きに進む)も含まれます(実物は次節の図2.2)。ですから $f'(x) = 0$ を解いて手に入るのは最小値の**候補のリスト**です。候補が複数なら、それぞれの $f$ を計算して比べれば決着がつきます。候補は方程式が絞ってくれるので、たいした手間ではありません。

——方程式さえ、解ければ。

## 2.2 「傾き = 0 を解けばいい」が通用しない関数が世の中のほとんど

前節の戦略を、少しずつ手強い相手に試します。階段を1段ずつ上がるつもりで見てください。

**1段目: 2次関数。** $f(x) = x^2 - 4x + 5$。$f'(x) = 2x - 4 = 0$ は一次方程式で $x = 2$。即決です。

**2段目: 4次関数。** $f(x) = x^4 - 2x^2$。微分すると $f'(x) = 4x^3 - 4x$。3次方程式ですが、

$$4x^3 - 4x = 4x(x-1)(x+1) = 0$$

と因数分解できるので、候補は $x = -1,\ 0,\ 1$。$f(-1) = -1$、$f(0) = 0$、$f(1) = -1$ なので、最小値は $x = \pm 1$ での $-1$($x = 0$ は前節で予告した「山頂」)。

```
 f(x)
   │＼                          ／
   │ ＼          ●            ／
 0 ┤  ＼       ／   ＼       ／       ● = 傾き 0 の点(3か所)
   │   ＼     ／      ＼    ／
−1 ┤     ●              ●
   └─────┬───────┬───────┬─────→ x
        −1       0       1
```

図2.2: $f(x) = x^4 - 2x^2$。傾きが 0 になる点は3つあり、両端の2つが谷底(最小値)、中央の1つは山頂。方程式 $f'(x) = 0$ は3つとも区別せずに「候補」として返してくる。

ここまでは戦略は順調です。

**3段目: 別の4次関数。** $f(x) = x^4 + x^3 - 2x + 1$。微分して

$$f'(x) = 4x^3 + 3x^2 - 2 = 0$$

今度の3次方程式は、うまい因数分解が見つかりません。3次方程式には「解の公式」が存在します(カルダノの公式)。ですがその公式は平方根と立方根が何重にも入れ子になった代物で、2次の解の公式のような気軽さはありません。4次にも公式はありますが、さらに悪化します。「原理的には解ける。しかし、もう紙と鉛筆の道具ではない」——この段から雲行きが怪しくなります。

**4段目: 6次関数。** $f(x) = \dfrac{x^6}{6} - x^4 + x^2 + 2x$。微分すると

$$f'(x) = x^5 - 4x^3 + 2x + 2 = 0$$

5次方程式です。ここで、はしごが外れます。**5次以上の方程式には、四則演算と根号で書ける解の公式が存在しません。** 「まだ見つかっていない」のではなく、「存在しない」と証明されています(アーベルとルフィニの定理。証明は本書の旅程の外なので結果だけ借ります)。因数分解できる幸運な例外を除けば、5次方程式を式変形で解く道は原理から閉ざされているのです。

そして**最終段: 私たちが本当に相手にする関数。** ラスボスの論文の Transformer は、$XW + b$ のような計算を何層にも入れ子に重ね、間に exp を含む関数(第4巻)を挟んだ巨大な合成関数です。動かせる変数——$W$ たちの成分——の総数は、原論文のモデルで約6500万個です。傾きにあたるものを 0 と置くと、6500万本の方程式が連立した系が現れます。一本一本が5次方程式どころではない複雑さで、です。これを式変形で解くという発想は、もはや成立しません。

階段を振り返ると、こうです。「$f'(x) = 0$ を解く」が通用するのは、2次関数や因数分解できる幸運な多項式といった、教科書の例題のために選ばれたような関数だけです。**世の中の関数のほとんど——そして機械学習に現れる関数のほぼすべて——では、この方程式は解析的に(analytically)、つまり式変形の有限の手順では解けません。**

2.1節の最後の一文が、ここで効きます。最小値探しは方程式に帰着した、ただし方程式さえ解ければ。その「さえ」が、いま崩れました。

## 2.3 だから反復で近づく、という発想の転換

行き詰まったときは、何に困っているのかを正確に言い直すのが定石です。

手に入らないのは、$f'(x) = 0$ の**解を一発で言い当てる式**です。では何なら手に入るのでしょうか。——**任意の点での $f'(x)$ の値**です。これは第1章の道具がそのまま使えます。$f'$ の式が書ければ代入すればよいし、式が手に負えなくても、数値微分 $\dfrac{f(x+h) - f(x-h)}{2h}$ は $f$ を2回計算するだけで動きます。方程式は解けないのに、傾きの**測定**はいつでもできます。この非対称性が突破口です。

しかも傾きの値は、ただの数ではありません。図2.1のとおり、**符号が「谷底がどっちか」を教えてくれる**のです。

- $f'(x) < 0$(下り坂)なら、谷底は右にある。右へ進むべき
- $f'(x) > 0$(上り坂)なら、谷底は左にある。左へ進むべき

いまいる場所で傾きを測れば、正解の座標はわからなくても、**正解のある方角**はわかります。だったらその方角へ少し動く。動いた先でまた傾きを測る。また動く。これを繰り返せば、一発で解けない谷底へじわじわ近づけるはずです。

これが、解析的に解く(closed-form で解を書き下す)のをやめて、**反復(iteration)で近づく**という発想の転換です。霧で何も見えない山中でも、足の裏は地面の傾きを感じ取れます。地図(=解の公式)は永遠に手に入らなくても、足裏の感覚だけを頼りに一歩ずつ下れば谷底にたどり着ける——この巻のタイトル「学習とは坂を下ること」の「下る」とは、この反復のことです。

本当にたどり着けるのか、2.2節で「解けない」と宣告された方程式で試します。$f(x) = \dfrac{x^6}{6} - x^4 + x^2 + 2x$ の傾き

$$f'(x) = x^5 - 4x^3 + 2x + 2$$

は5次方程式としては解けませんが、値の計算はただの代入です。まず2点で測ります。

$$f'(-2) = -32 + 32 - 4 + 2 = -2 < 0, \qquad f'(-1) = -1 + 4 - 2 + 2 = +3 > 0$$

$x = -2$ では下り坂(谷底は右)、$x = -1$ では上り坂(谷底は左)です。つまりこの2点に**挟まれた区間のどこか**に、傾きが 0 になる谷底があります。なら区間の真ん中で傾きを測れば、谷底が真ん中より左か右かがわかり、区間を半分に絞れます。繰り返します。

| 反復 | 注目区間 | 真ん中 $x$ | $f'(x)$ の値 | 結論 |
|---|---|---|---|---|
| 1 | $[-2,\ -1]$ | $-1.5$ | $+4.91$(上り) | 谷底は左半分 |
| 2 | $[-2,\ -1.5]$ | $-1.75$ | $+3.52$(上り) | 谷底は左半分 |
| 3 | $[-2,\ -1.75]$ | $-1.875$ | $+1.44$(上り) | 谷底は左半分 |
| 4 | $[-2,\ -1.875]$ | $-1.9375$ | $-0.09$(下り) | 谷底は右半分 |
| 5 | $[-1.9375,\ -1.875]$ | … | … | … |

4回の反復で、谷底の居場所は幅 $1$ の区間から幅 $0.0625$ の区間 $[-1.9375,\ -1.875]$ まで絞り込まれました(4回目の傾き $-0.09$ は 0 にかなり近く、谷底が目前であることまで匂わせています)。反復のたびに区間は半分になるので、10回で幅は約 $1/1000$、20回で約 $1/100$万です。解の公式が**存在しない**と証明された方程式が相手でも、足裏の感覚——各点での傾きの符号——さえ測れれば、谷底はいくらでも追い詰められます。

種明かしをすると、この「真ん中で測って半分に絞る」は反復のいちばん素朴な実演で、見せたかったのは絞り込みのテクニックそのものではありません。受け取ってほしいのは、3つの転換です。

第一に、**答えを「解く」対象から「近づく」対象に変えた**ことです。出てくるのは $x = -1.93\ldots$ という近似値であって、根号で書けた厳密解の顔はしていません。しかし私たちの目的は $W$ を「良く」することでした。谷底ぴったりと、谷底から $1/100$万 ずれた点とで、$f$ の値は実用上区別がつきません。**厳密解への未練を捨てると、解ける問題の範囲が爆発的に広がる**——これがこの章の主役の交代劇です。

第二に、答えが**手順(アルゴリズム)の形**をしていることです。「この式に代入せよ」ではなく「測って、動いて、繰り返せ」です。だから関数がどれほど複雑でも、傾きの値さえ計算できれば同じ手順がそのまま回ります。手順は関数の中身に立ち入らないのです。

第三に、新しい登場人物として**反復の回数**が現れたことです。解の公式の世界に「何回目」はありませんでしたが、反復の世界では何歩めにいるかが精度を決めます。この「何歩め」は、ラスボスの式で $step\_num$ という名で堂々と登場します。あの式が読める日まで、覚えておいてください。

ただし、上の表の手順をそのまま機械学習に使うわけではありません。「区間に挟んで絞る」は変数が1個だから成立する芸当で、変数が6500万個になると「挟む」ことができなくなります(なぜできないかは、変数が2個になる第4章で見通しがよくなります)。次章では、挟み撃ちの代わりに、**いまいる一点の傾きだけを見て、傾きの大きさに応じた歩幅で坂を下る**——どんな変数の数にもそのまま伸びる、たった1行の反復手順を手に入れます。それが勾配降下法、この巻の看板です。

## まとめ

- 最小値の足元では傾きが 0 になる。よって最小値探しは「方程式 $f'(x) = 0$ を解く」に帰着する——ただし傾き 0 の点は谷底とは限らず(山頂・踊り場もある)、方程式が返すのは候補のリスト
- その方程式が解析的に解けるのは、低次の多項式や因数分解できる幸運な例だけ。5次以上の方程式には解の公式が存在しないことが証明されており、まして機械学習の関数は数千万変数の連立になる。**世の中は解けない方程式だらけ**
- 一方、方程式は解けなくても、**任意の点での傾きの値**はいつでも計算できる(第1章の道具)。そして傾きの符号は「谷底がどっちか」を教える
- だから、一発で解くのをやめて反復で近づく。測って、動いて、繰り返す。出てくるのは近似値だが、私たちの目的にはそれで足りる
- 反復の世界では「いま何歩めか」が意味を持つ。この概念は論文で $step\_num$ という名で再会する

**ラスボスとの距離**: Section 5.3 の lrate の式に $step\_num$(反復の何歩め)が入っている理由——学習が「一発で解く」のではなく「反復で近づく」手続きだから——の土台ができました。歩き方そのもの(勾配降下法)は次章です。

## 演習

**演習1** 次の (a)〜(e) について、$f'(x) = 0$ という方程式を立て、「解析的に解ける」「原理的には解けるが実用的でない」「解析的には解けない」のどれかに仕分けてください。解けるものは解き、最小値を求めてください。

$$\text{(a)}\ f(x) = x^2 - 6x + 10 \qquad \text{(b)}\ f(x) = x^4 - 8x^2 \qquad \text{(c)}\ f(x) = x^4 - x^3 + x$$

$$\text{(d)}\ f(x) = \frac{x^6}{6} + \frac{x^4}{4} \qquad \text{(e)}\ f(x) = \frac{x^6}{6} - x^4 + x + 1$$

<details><summary>略解</summary>

(a) $f'(x) = 2x - 6 = 0$。一次方程式で**解ける**。$x = 3$、最小値 $f(3) = 1$。

(b) $f'(x) = 4x^3 - 16x = 4x(x-2)(x+2) = 0$。3次だが因数分解できるので**解ける**。候補は $x = -2, 0, 2$ で、$f(\pm 2) = -16$、$f(0) = 0$。最小値は $x = \pm 2$ での $-16$($x=0$ は山頂)。

(c) $f'(x) = 4x^3 - 3x^2 + 1 = 0$。3次方程式だが、うまい因数分解が見つからない(有理数の解がない)。カルダノの公式があるので**原理的には解けるが実用的でない**。

(d) $f'(x) = x^5 + x^3 = x^3(x^2 + 1) = 0$。5次だが因数分解でき、実数の解は $x = 0$ のみ($x^2 + 1 = 0$ に実数解はありません)なので**解ける**。最小値は $f(0) = 0$。——5次以上でも、因数分解できる幸運な例外は解けます。「次数が高い = 解けない」ではなく、「一般には解の公式がない」が正確な仕分け基準です。

(e) $f'(x) = x^5 - 4x^3 + 1 = 0$。5次で、因数分解の当てもない(有理数の解がない)ので、**解析的には解けない**。2.3節の反復なら谷底に近づけます。

</details>

**演習2** $f(x) = x^3 - 3x^2 + 3x$ について $f'(x) = 0$ を解き、得られた点が最小値かどうかを、その点の左右での傾きの符号を調べて判定してください。

<details><summary>略解</summary>

$f'(x) = 3x^2 - 6x + 3 = 3(x-1)^2 = 0$ なので、傾き 0 の点は $x = 1$ のみ(方程式は解析的に解けます)。左右の傾きは $f'(0) = 3 > 0$、$f'(2) = 3 > 0$ で、**両側とも上り坂**。つまり $x = 1$ は、上り坂の途中で一瞬だけ平らになる「踊り場」であって、最小値ではありません(実際、$x$ を小さくすれば $f$ はいくらでも小さくなります)。方程式が解けることと、最小値が手に入ることは別問題——傾き 0 はあくまで候補の条件です。

</details>

**演習3** 中身を見せてもらえない関数 $f$ があります。第1章の数値微分で傾きを測ったところ、$f'(1) = -5$、$f'(2) = -1$、$f'(3) = +2$ でした。

(a) 谷底はどの区間にありそうですか。
(b) いま $x = 1$ に立っているなら、左右どちらへ動くべきですか。
(c) 「傾きの値を見て動く方向を決める規則」を1行で書いてください。

<details><summary>略解</summary>

(a) 傾きの符号が $x = 2$(負)と $x = 3$(正)の間で反転しているので、谷底は区間 $(2,\ 3)$ にありそうです。(b) $f'(1) = -5 < 0$ は下り坂なので、**右**へ。(c) 「傾きが負なら右へ、正なら左へ」——まとめれば、**傾きの符号と逆の方向へ動く**。この規則を「どれだけ動くか」まで含めて1行の式にしたものが、次章の勾配降下法です。

</details>
