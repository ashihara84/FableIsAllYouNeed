## 0.1 ここまでの到達点 — 式(1)は読めた。だが図1には、まだ箱がある

第5巻です。第4巻の終章で、このシリーズ初の出来事が起きました。論文の式(1)——

$$\mathrm{Attention}(Q, K, V) = \mathrm{softmax}\!\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

——の**すべての記号が読める**ようになったのです。$Q$、$K$、$V$、$QK^T$、転置は第1巻。softmax と $\sqrt{d_k}$ は第4巻。第1巻の序章で「おそらく、何ひとつ読めないはずです」と掲げたあの式を、いまや shape から分散の議論まで含めて説明できます。折り返し地点として、これ以上ない戦果です。

次はどこへ向かうのか。

論文3ページめの**図1**——Transformer のアーキテクチャ図、2つの塔が並び中に箱が積み上がった、あの有名な図を見てください。箱のラベルを上から読み上げます。Softmax。Linear。**Add & Norm**。**Feed Forward**。Add & Norm(また出てきました)。Multi-Head Attention。Positional Encoding。Embedding。

このうちいま読めるのは、第4巻で倒した Softmax、第1巻からの付き合いの Linear、そして中身の式(1)が読める Multi-Head Attention の箱(「Multi-Head」が何かはまだですが)。

残るは **Feed Forward** と、2回登場する **Add & Norm**。さらに本文には、図にない部品——**Dropout**——も待っています。

先に宣言します。**この巻を終えると、図1の attention 以外の箱が、全部読めるようになります。**

## 0.2 ラスボスの掲示 — 部品は読めるのに、意図が読めない

今回のラスボスに会いに行きましょう。3箇所から引きます。

> $$\mathrm{FFN}(x) = \max(0,\ xW_1 + b_1)W_2 + b_2$$
> — Vaswani et al., "Attention Is All You Need", Section 3.3, 式(2)
>
> *"We employ a residual connection around each of the two sub-layers, followed by layer normalization. That is, the output of each sub-layer is LayerNorm(x + Sublayer(x))."*
> — 同論文, Section 3.1
>
> *"We apply dropout to the output of each sub-layer, before it is added to the sub-layer input and normalized."*
> — 同論文, Section 5.4

英文を訳します。Section 3.1 は「2つの各サブ層のまわりに residual 接続を施し、その後に層正規化を行う。すなわち、各サブ層の出力は LayerNorm(x + Sublayer(x)) である」。Section 5.4 は「各サブ層の出力に dropout を適用する。サブ層の入力に加算され正規化される前に、である」。

今回の「読めなさ」は、これまでの巻と手触りが違います。

式(2)をよく見てください。$xW_1 + b_1$ は、第1巻からずっと使ってきた linear そのもの。その結果に $W_2$ を掛けて $b_2$ を足す——linear をもう1回。つまりこの式は、**行列計算としての骨格なら、すでに読めてしまう**のです。

読めないのは、真ん中に鎮座する $\max(0,\ \cdot)$ だけ。「0 と比べて大きいほうを取る」という拍子抜けするほど単純な操作が、なぜ式の中心にいるのか。そもそも、なぜ linear を2回重ねるのか。第1巻5.3で確認したとおり、**線形変換の合成は1つの線形変換にまとめられる**のでした。$W_1$ と $W_2$ を2枚用意しても1枚と同じことしかできないはず——では何のために2枚使うのか。

その答えの鍵こそが $\max(0,\ \cdot)$ です。部品はぜんぶ知っているのに、設計の意図がまったく読めない。それが式(2)の読めなさです。

Section 3.1 と 5.4 の文章は、もっと率直に読めません。residual connection、sub-layer、layer normalization、dropout——単語が軒並み初耳です。そしてこれらは、図1で attention の箱を上下から挟む「Add & Norm」の正体でもあります。

この巻を通して、これらの部品を1つずつ——なぜ必要なのか、という需要から——手に入れていきます。

## 0.3 この巻のもう一つの主役 — もう手で微分したくない

この巻には、上のラスボス引用のどこにも写っていない、もう1人の主役がいます。

ここまで、勾配が必要になるたびに何をしてきたか。第3巻では線形回帰の損失 $L$ をパラメータで偏微分し、$\partial L / \partial w$ を**手で**導出。第4巻では sigmoid + log loss の勾配を**手で**導出し、$(y - t)\,x$ という美しい形に辿り着きました。softmax + cross-entropy でも、また**手で**。

毎回、手でやってきたのです。そして毎回、それなりにしんどかったはずです。

この巻で作るニューラルネットワークは、linear と活性化関数を交互に何層も重ねます。パラメータは $W_1, b_1, W_2, b_2, \ldots$ と層の数だけ増える。この構造に対して $\partial L / \partial W_1$ を手で導出すると、どうなるか——第2章で一度だけ実際にやってもらいます。先に予告すると、地獄です。そしてその地獄を見たあとで約束します。**それがあなたの最後の手導出になります。**

微分を、自動化します。

土台は**誤差逆伝播(backpropagation)**。第2巻5章で手に入れた連鎖律と計算グラフが、この巻の第3章で主役として帰ってきます。そして第4章で最大の山に挑みます——**自動微分(automatic differentiation)のエンジンを、自分の手で作る**のです。

規模は100行ちょっとです。値と勾配を覚える小さなクラスに、足し算・掛け算・$\max(0,\ \cdot)$ などの演算を1つずつ生やしていく。完成したエンジンに第3巻の回帰と第4巻のロジスティック回帰を解かせると、手導出したあの勾配たちが**すべて自動で**出てきます(第5章)。これは、PyTorch で `loss.backward()` と書いたときに起きていることの正体そのものでもあります。構造は同じで、規模が違うだけです。

ラスボスの部品(FFN、residual、layer norm、dropout)を揃える巻であると同時に、残り3巻すべてを支える計算エンジンを鍛造する巻。それが第5巻です。

## 0.4 この巻で扱わないこと

恒例の白状コーナーです。今回は3つ。

1つめ。**CNN(畳み込みニューラルネットワーク)を扱いません。** ニューラルネットの教科書なら間違いなく1章を割く話題です。しかし論文の Abstract はこう言っています——"dispensing with recurrence and convolutions entirely"(再帰と畳み込みを完全に排して)。畳み込みは、この論文が「捨てる」と宣言した道具。捨てられる側を学んでから捨てるほど、私たちの道のりは長くありません。

2つめ。**RNN(再帰型ニューラルネットワーク)も、この巻では扱いません。** こちらは事情が違います。RNN は同じ Abstract で捨てられているのに、第6巻でしっかり登場します。「attention だけでいい」というタイトルの意味を理解するには、RNN の痛み——並列化できない、遠くを覚えられない——を自分で体感する必要があるからです。つまり RNN は、必要になってから学ぶ。いつもの需要駆動です。この巻で作るのは、時間や順序の概念を持たない、いちばん素直な多層パーセプトロンだけです。

3つめ。**重みの初期化の理論を、一般論としては展開しません。** 深いネットワークでは「最初の乱数の選び方」が思いのほか効くのですが、Xavier 初期化に代表される体系的な理論には立ち入らず、第6章で**使う分だけ**触れます。そこで行う計算は、第4巻7章で $\sqrt{d_k}$ を導いたのと同じ分散の計算です。あの道具がもう一度だけ働きます。

## 巻頭付録 論文読解マップについて

シリーズ共通の巻頭付録、**論文読解マップ**(`paper-map.md`)で現在地を確認しておきましょう。

第4巻までに「読める」側へ移ったのは、式(1)の全記号、Section 5 の training と batching の語彙、learning rate の式の意図でした。地図の上では、Section 3.2(attention の心臓部)と Section 5 の一部に旗が立っている状態です。

今回は、Section 3 の残りの平野部——3.3 の FFN と、3.1 後半の LayerNorm(x + Sublayer(x))——に加えて、Section 5.4 の dropout に旗を立てにいきます。第5巻が担当するのは、地図の中の次の行です。

| 論文の箇所 | 内容 | この巻の章 |
|---|---|---|
| 3.3 式(2) | $\mathrm{FFN}(x) = \max(0,\ xW_1 + b_1)W_2 + b_2$、ReLU、$d_{ff} = 2048$ | 第1・2章 |
| 3.1 | $\mathrm{LayerNorm}(x + \mathrm{Sublayer}(x))$(residual + layer norm) | 第6章 |
| 5.4 | dropout($P_{drop} = 0.1$) | 第6章 |

この巻を終えると、図1のアーキテクチャ図は「読める箱」が多数派になります。残るは Multi-Head Attention、Positional Encoding、Embedding——いずれも第6巻以降の獲物です。

それでは始めましょう。最初の問いはこれです——第1巻からずっと頼ってきた `X @ W + b` に、実は**どうしてもできないこと**があります。それは何でしょうか。第1章では、まずその限界に正面からぶつかります。
